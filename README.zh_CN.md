## 安装

使用 `pnpm`

```bash
pnpm add @tomjs/vite-plugin-electron -D
```

使用 `yarn`

```bash
yarn add @tomjs/vite-plugin-electron -D
```

使用 `npm`

```bash
npm i @tomjs/vite-plugin-electron --save-dev
```
